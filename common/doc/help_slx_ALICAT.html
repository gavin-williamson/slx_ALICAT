Help page for blockset ALICAT_Controller

Description:

The blocks in this library can be used read and control ALICAT devices in real time. To use these blocks for their inteded purpose, the Simulink model must be set-up in a particular manner.
1) Set Stop Time to "inf"
2) Enable pacing to slow down simualtion and set to 1. This setting can be found in the drop down below the "Run" button.
3) Under "Solver" in "Model Settings" change the type to "Fixed-step" and then change the "Fixed-step size" to your desire time-step.
This set up enables Simulink to run continuously and in a pseudo-real-time manner. Since this is not a standard Simulink application, the simulation will not run a perfect real time manner keeping an exact step size, but Simulink overall does a decent job of maintaining a consistent time-step. Of course this does depend on the complexity of your model and your desired time-step. It will also notify you (in the bottom left corner) if the simulation pacing is falling.
